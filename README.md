🌦️ Weather Web App

A simple and responsive Weather Web Application that allows users to search for weather information for any city. The application displays real-time weather data such as temperature, humidity, wind speed, and weather conditions.

This project is built using HTML, CSS, and JavaScript and uses a weather API to retrieve live weather data. Weather apps typically fetch real-time weather information from external APIs and display details like temperature, humidity, and wind speed for a selected location.

🚀 Features

🌍 Search weather by city name

🌡️ Display current temperature

💨 Show wind speed

💧 Show humidity

☁️ Display weather condition

📱 Responsive design for mobile and desktop

⚡ Fast and simple user interface

🛠️ Technologies Used

HTML5 – Structure of the web page

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Application logic

Weather API – Open-Meteo API (https://api.open-meteo.com/v1/forecast?latitude)

⚙️ Installation & Setup

Clone the repository

git clone https://github.com/Vimukthi26/Weather-Web-App.git

Open the project folder

cd Weather-Web-App

Open index.html in your browser.

💻 How to Use

Open the application in your browser.

Enter a city name in the search box.

Click the search button.

The app will display the current weather information for that location.
